Plataforma E-Commerce B2B Mayorista

El Desafío

Nuestro cliente, una distribuidora nacional, necesitaba digitalizar su proceso de ventas mayorista. Los pedidos se realizaban por teléfono, lo que generaba errores de stock y retrasos.

La Solución

Desarrollamos una plataforma web progresiva (PWA) utilizando React para el frontend y Node.js para el backend. La clave fue la integración en tiempo real con su ERP antiguo mediante una API intermedia personalizada.

Resultados Clave

  • Reducción del 40% en llamadas de gestión de pedidos.
  • Sincronización de stock en tiempo real (< 200ms).
  • Aumento del 25% en ventas recurrentes el primer trimestre.
