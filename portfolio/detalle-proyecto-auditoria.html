Blindaje Corporativo Sector Banca

El Desafío

La entidad financiera requería actualizar su infraestructura perimetral para cumplir con las nuevas normativas ISO 27001 y protegerse contra ataques DDoS persistentes.

La Solución

Implementamos una arquitectura Zero Trust. Desplegamos firewalls de nueva generación (NGFW) de Fortinet y configuramos segmentación de red avanzada para aislar activos críticos.

Tecnologías Usadas

Fortinet Security Fabric, Cisco Identity Services Engine (ISE), SIEM para monitorización 24/7.
